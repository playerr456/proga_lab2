# Sequence Container Library

## proga_lab2

Данный проект реализует иерархию классов для работы с последовательностями данных в C++. Основное внимание уделено двум фундаментальным структурам: динамическим массивам и связным спискам, а также их мутабельным (изменяемым) и иммутабельным (неизменяемым) версиям.

## Особенности реализации

- **Интерфейс Sequence**: Общий интерфейс для всех типов последовательностей, определяющий основные операции (`Append`, `Prepend`, `InsertAt`, `Get`, `GetSubSequence`, `Concat`).
- **Контейнеры**:
    - `DynamicArray`: Динамический массив с автоматическим изменением размера.
    - `LinkedList`: Двусвязный список.
- **Мутабельность**:
    - `MutableArraySequence` / `MutableListSequence`: Операции изменяют текущий объект.
    - `ImmutableArraySequence` / `ImmutableListSequence`: Операции возвращают новый объект, оставляя исходный неизменным.
- **Паттерн Iterator (Enumerator)**: Реализован механизм перебора элементов через интерфейсы `IEnumerable` и `IEnumerator`.
- **Шаблоны (Templates)**: Код полностью шаблонизирован, что позволяет работать с любыми типами данных.

## Структура проекта

- `Sequence.h`: Базовый интерфейс последовательности.
- `ArraySequence.h` / `.tpp`: Реализация последовательности на базе массива.
- `ListSequence.h` / `.tpp`: Реализация последовательности на базе списка.
- `DynamicArray.h` / `.tpp`: Внутренний класс динамического массива.
- `LinkedList.h` / `.tpp`: Внутренний класс связного списка.
- `IEnumerable.h` / `IEnumerator.h`: Интерфейсы для итерации.
- `main.cpp`: Консольный интерфейс для демонстрации работы.
- `tests.cpp`: Набор модульных тестов для проверки корректности.

## Сборка и запуск

### Требования
- Компилятор C++ (поддерживающий стандарт C++11 и выше, например `g++` или `clang`).

### Сборка через g++
Для сборки основной программы:
```bash
g++ main.cpp -o main.exe
```

Для сборки тестов:
```bash
g++ tests.cpp -o tests.exe
```

### Запуск
После сборки запустите исполняемый файл:
```bash
./main.exe
```

## Тестирование
Для проверки работоспособности всех компонентов запустите тесты:
```bash
./tests.exe
```
Программа выполнит серию `assert` проверок и выведет `ALL TESTS PASSED SUCCESSFULLY` в случае успеха.
